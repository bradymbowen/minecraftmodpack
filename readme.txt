How to install the modpack:

1.) Install the latest version of Forge for 1.12.2: version 1.12.2 - 14.23.5.2847)
    http://files.minecraftforge.net/maven/net/minecraftforge/forge/index_1.12.2.html
2.) In the official Minecraft launcher, create a new installation with version "release 1.12.2-forge1.12.2-14.23.5.2847" and choose a different game directory (required if you want your old minecraft versions to still work)
3.) Run this version once to populate the new folder, and then close the game.
4.) Under Edit Installation (click on the ... that appears when you hover over the installation, click on More Options and under JVM Arguments change -Xmx2G to -Xmx4G or higher (the number in the command may look different)
5.) Download and move the "mods" and "config" folders to the base directory.
6.) Should be done, ask me if any error occurs
7.) This mod is very beta in crafting and mod interconnectivity
